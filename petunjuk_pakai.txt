PETUNJUK PAKAI BRO
=========================================================

Untuk menyinkronkan repositori lokal dan jarak jauh, pertama-tama kita mengambil riwayat perubahan dari repositori jarak jauh dan menggabungkannya ke dalam repositori lokal menggunakan perintah pull. Kemudian kami mendorong riwayat perubahan lokal kami ke repositori jarak jauh menggunakan push perintah

Untuk menarik dari repositori jarak jauh dan bergabung ke repositori lokal kami menggunakan perintah Git:

git pull origin master --allow-unrelated-histories <GIT COMMAND>

Bendera --allow-unrelated-histories diperlukan saat pertama kali kami menarik karena kami ingin menggabungkan dua repositori yang dibuat secara terpisah dan saat ini tidak berbagi riwayat terkait.

Setelah pertama kali, pull perintah git tidak memerlukan bendera itu:

git pull origin master <GIT COMMAND>
